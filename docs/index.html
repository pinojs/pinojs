the super fast, all natural JSON logger

Pino is an extremely fast node.js logger, inspired by Bunyan. Logging is a necessity, but it shouldn't be what your appliction spends all of its time doing. When you use Pino for your logger you save orders of magnitude more time than with other loggers:

logger.info('hello %s %j %d', 'world', {obj: true}, 4, {another: 'obj'})

benchDebug*10000: 640.001ms
benchBunyan*10000: 2888.825ms
benchWinston*10000: 2616.285ms
benchBole*10000: 1313.470ms
benchLogLevel*10000: 1487.610ms
benchPino*10000: 486.367ms

Getting started with Pino is easy. Just npm install --save pino and start writing: 


        var pino = require('pino')
        var log = pino({level: 'debug'})

        log.info('a simple info message')
        log.debug('interesting object: %j', {foo: 'bar', pino: 'is a way of life'})
